# RareAlert Chatbot

RareAlert is an intelligent chatbot designed to assist users in identifying potential rare diseases based on symptoms and basic medical inputs. The system analyzes user-provided information to suggest possible conditions, recommend treatment directions, and guide users toward appropriate medical specialists.

To improve accessibility and user experience, RareAlert supports multilingual interaction by translating conversations into the userâ€™s preferred language using Google Translate. The application features an interactive and easy-to-use interface built with Streamlit.

# Key Features

Predicts potential rare diseases based on user-reported symptoms

Provides preliminary treatment guidance and specialist recommendations

Supports multiple languages for broader accessibility using googletrans

User-friendly and interactive interface developed with Streamlit
